# 准备环境（数据库）

目标：让后端能稳定连上本机 MySQL，并具备一个“干净可用”的业务数据库（schema）和账号。

## 适用范围

- macOS
- 本机 MySQL（推荐 5.7；如果你本机是 8.0 也可按本指南继续）
- 你已安装数据库连接工具（推荐 DBeaver）
