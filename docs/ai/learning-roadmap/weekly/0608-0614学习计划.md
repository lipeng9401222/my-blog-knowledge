# 0608-0614 学习计划：环境搭建与 Python 基础

## 本周定位

这是 36 周路线的第 1 周，重点不是“学完 Python”，而是把后续学习都会用到的环境、目录、脚本运行方式和记录习惯建立起来。

本周目标：跑通 Python/Git/编辑器环境，并完成一个设备温度报警统计脚本。

## 本周目标

| 目标 | 具体标准 |
|---|---|
| 环境可用 | 能在命令行运行 `python`、`pip`、`git` |
| 编辑器可用 | 能创建项目目录、编写 `.py` 文件并运行 |
| 语法入门 | 掌握变量、列表、字典、条件、循环、函数 |
| 文件处理 | 能读取 CSV，输出温度统计和报警数量 |
| 学习习惯 | 写一份本周 README 或学习日志 |

## 每日安排

## 本周交付物

| 文件 | 要求 |
|---|---|
| `hello.py` | 输出 `Hello Industrial AI` |
| `basic_types.py` | 用字典和列表表达设备传感器记录 |
| `rule_alarm.py` | 按温度阈值输出报警等级 |
| `alarm_utils.py` | 封装报警判断和统计函数 |
| `main.py` | 调用函数并输出统计结果 |
| `sample_equipment_data.csv` | 至少包含 5 条设备数据 |
| `README.md` | 说明项目用途、目录结构、运行命令和复盘 |

## 报警规则

| 条件 | 等级 |
|---|---|
| `temperature < 70` | `normal` |
| `70 <= temperature < 80` | `notice` |
| `80 <= temperature < 90` | `warning` |
| `temperature >= 90` | `critical` |

## 验收标准

```bash
python --version
pip --version
git --version
python week01/hello.py
python week01/main.py
```

验收通过时，应该满足：

1. 能看到 Python、pip、Git 版本号。
2. `hello.py` 能成功输出 `Hello Industrial AI`。
3. `main.py` 能读取 CSV 并输出温度最大值、最小值、平均值和报警数量。
4. README 能说明项目用途、目录结构、运行命令和本周复盘。
