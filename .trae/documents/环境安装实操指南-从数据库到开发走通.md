## Verification Steps (Execution Phase)

执行阶段每完成一段就做一次可重复验证：
1. 数据库：
   - `mysql -uroot -p -e "select 1"` 或 DBeaver 执行 `select 1`
   - 新库存在、字符集正确、用户权限正确
2. 后端：
   - `tomcat7:run` 启动无报错
   - `http://localhost:8080/epoint-web` 可访问（或至少返回登录/授权页）
   - 数据库表自动创建完成（对比“访问系统”章节截图/SQL）
3. 前端：
   - `pnpm run dev` 启动成功
   - 前端页面可访问，关键接口通过代理到后端（Network 里无 404/502）
4. 文档：
   - `npm run build`（构建知识库站点）通过，保证新增文档不会破坏站点发布
