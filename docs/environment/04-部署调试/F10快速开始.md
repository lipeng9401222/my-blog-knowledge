# F10框架快速开始

## 部署包启动

部署包下载：[epoint-web.war](https://192.168.203.10:81/F10/%E9%83%A8%E7%BD%B2%E5%8C%85/epoint-web.war)

下载后 直接放入本地tomcat/webapps目录，找到tomcat/bin/startup.bat双击运行启动：

执行到显示验证码授权日志即启动成功

![image-20260305164144871.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=3cf1e86e-ea02-48a2-b02e-bb9e980b3608)

访问地址验证：http://localhost:8080/epoint-web/home

首次访问会跳转到授权码注册界面，根据提示申请注册码上传即可：

![image-20260305164322660.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=327c3bae-9295-4ead-9589-505108f6ec64)

首次注册成功后会跳转到F9登录页，可以再次访问http://localhost:8080/epoint-web 跳转F10新版登录页，输入默认账户admin/11111登录

![image-20260305171029192.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=ab209f75-ebe3-4cae-ad16-75fdd518f536)

登录后进入首页可以看到默认的客户端Eva主题首页

![image-20260305164608605.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=07e11457-c496-4205-bb2f-fb7a5ddd4a7a)

访问http://localhost:8080/epoint-web/admin 可进入后台管理

![image-20260305181015786.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=b8aee049-0d33-4b45-9e59-c84246242e90)

访问http://localhost:8080/epoint-web/egoapp 可进入ego低码平台

![image-20260305181001206.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=92459c4c-c5ef-4998-9f5c-ad03d1c8428)

也可以在右上角头像选择菜单进入：

![image-20260305173913459.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=47f82407-9f9c-48f8-a7f-5c77f2663f8)

## 开发包启动

如果本地已有开发环境（maven+jdk1.8），也可使用开发包启动

开发包下载：[[epoint-web.zip]](?file=assets\012\090\040\epoint-web.zip)

下载后解压出来，使用IDEA打开maven工程

![image-20260306084427622.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=b8ced6fe-2722-4de6-92b4-382379239499)

![image-20260306084459516.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=d813e158-b03b-442e-9590-b00787fa7f67)

使用tomcat插件启动

![image-20260306084815320.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=49bba600-41e8-4600-ba9e-86793d59f3d)

启动成功后可以访问http://localhost:8080/epoint-web/home验证，界面效果不再赘述。
