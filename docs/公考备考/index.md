# 公考备考

收录公务员考试的备考方案、模块突破策略与课程匹配手册。以数据化复盘、分阶段规划和高性价比提分为核心，服务在职备考场景。

## 已收录

- [江苏省考 B 类一战上岸备考方案（在职）](/公考备考/江苏省考B类一战上岸备考方案) — 考情全景、行测提分优先级、申论方法、5-6 个月四阶段规划与复盘机制。
- [模块专项突破与课程匹配手册](/公考备考/模块专项突破与课程匹配手册) — 行测五大模块 + 申论的题型拆解、推荐老师课程与每日练习量。

## 阅读路径建议

1. 先读《备考方案》建立整体节奏与岗位选择策略。
2. 再按《课程匹配手册》逐模块选课、定每日练习量。
3. 全程用错题本（四分类）+ 模考追踪表做数据化复盘。
