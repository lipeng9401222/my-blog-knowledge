# 08 每周检查清单

## 使用方式

每周结束时复制本周清单到自己的学习日志里，逐项打勾。不要追求所有项都完美，但必须每周有产出。

打勾规则：

| 标记 | 含义 |
|---|---|
| `[x]` | 已完成 |
| `[ ]` | 未完成 |
| `[~]` | 部分完成 |

## 每周固定复盘模板

```markdown
# 第 X 周复盘

## 本周目标

## 本周完成

- [ ] 学习：
- [ ] 代码：
- [ ] 文档：
- [ ] 项目：

## 本周产出链接

## 遇到的问题

## 下周最重要的一件事
```
