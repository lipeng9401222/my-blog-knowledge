# 实操指南

这套文档专门面向 macOS，本机 MySQL（推荐 5.7，当前也支持按 MySQL 8.0 路线走），后端使用 Tomcat 插件（war）方式启动，并包含 Vue 组件化前端启动。目标是从 0 到 1 走通“数据库 → 后端 → 前端 → 访问登录页”全链路。

## 你会得到什么

- 每一步都有明确的目的、操作、预期结果、失败排查
- 每一步产出都沉淀成文档，便于复用和回滚
- 全流程结束后，你可以在浏览器看到可登录的系统页面（并能定位常见问题）

## 推荐执行顺序

1. [准备环境](./准备环境/)：确认 MySQL、创建数据库与账号、验证连接
2. [开发环境启动](./开发环境启动/)：拉取后端代码并用 Tomcat 插件启动；启动 Vue 组件化前端并配置代理
3. [访问系统](./访问系统/)：验证表初始化、访问地址、授权/登录与排查

## 执行约定（重要）

- 所有路径建议使用英文 + 无空格（避免工具链兼容问题）
- 所有密码不要写进仓库文档：文档里只写占位符，你在本机执行时替换
- 每一步做完，把“你实际得到的输出”补充到对应文档的“本步产出”小节
