# 性能优化

Web 性能优化技巧，包括加载优化、渲染优化、缓存策略与指标治理。

## 推荐阅读

- [Core Web Vitals 实战：从指标理解到前端性能优化闭环](/performance/core-web-vitals-practice)

## 适合优先沉淀的内容

- Core Web Vitals 指标解释与治理流程
- 资源加载优化：图片、脚本、字体与缓存
- 渲染性能优化：长任务拆分、列表渲染、交互响应
- 工程侧保障：性能预算、自动化巡检、上线回归

## 阅读建议

如果你是从业务项目出发，建议先从真实指标和用户反馈倒推优化，不要一开始就陷入“把所有优化点都做一遍”的低效方式。
