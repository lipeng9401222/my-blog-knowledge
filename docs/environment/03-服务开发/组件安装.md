## 如何安装新组件

- 发布的框架默认集成了一些常用组件，如果你所需要的组件没有集成，那么只需要知道组件坐标，将其加入pom中即可
- 在安装的时候，建议也和框架出厂保持一致，在上方properties中定义组件版本号，下面dependencies中关联，这样后续要升级组件版本，也只需要修改上面属性定义中的值即可。
- 下面我们以安装大屏设计器为例来讲解下详细过程

- 1、首先打开[[框架组件清单](https://fdoc.epoint.com.cn/onlinedoc/rest/d/2aIjQ3 "框架组件清单")](?file=010-项目介绍/010-产品介绍#框架组件)，找到大屏设计器组件

![20240809-95c1ddcb.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=c292e746-6686-414b-9028-44af511e7b5e)

- 2、找到组件的坐标，组件坐标在文档上有，可以直接拷贝使用，也可以点击链接进入到对应组件内，然后打开部署指南，找到所需版本坐标进行集成

![20240809-df39da62.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=5ea63d8e-d2b6-478b-80b9-efcf8e095a25)

- 3、打开`pom.xml`文件，添加属性

```xml
<!-- 大屏 -->
<epoint-bigscreen-version>1.1.2</epoint-bigscreen-version>
```

![20240809-6bdd6003.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=89da3d33-f539-41d1-b2cf-fca3fe9dc7b0)

- 4、继续在`pom.xml`中添加依赖

```xml
<dependency>
    <groupId>com.epoint.frame</groupId>
    <artifactId>epoint-bigscreen</artifactId>
    <version>${epoint-bigscreen-version}</version>
</dependency>
```

- 5、重启系统，可以看到大屏设计器相关功能菜单已经自动注册

![20240809-43d58467.png](https://fdoc.epoint.com.cn/onlinedoc/rest/frame/base/attach/attachAction/getContent?isCommondto=true&attachGuid=1c6d6c0e-0e0c-4c9a-aadb-f3911d42db40)
